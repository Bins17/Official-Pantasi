Rules & Format

A. Updated Computation Formula

x 1 per Point

x 1.5 per Assist

x 2 per Steal

x 2 per Block

x 0.5 per 3 Point FG made

- 1 per Turnover

B. Quadro Cup Basic Format

8 Maximum Cancels (5man).

10 Maximum Cancels (6man).

C. Basic Rules

1. All sets must be sent in FB Messenger and FB Group, sending should only start on the provided submission time.

2. Each Quadro Team can have only 1 (E) Emergency set.

3. Salary used in Lineup should only be 305k at maximum for 5man lineups, and 350k maximum for 6man lineups.

4. All sets must be sent before the first NBA game begins, usually 10 minutes after the scheduled time of the game it starts.

5. Only 1 team member can send their team's lineup, the 4 lineups should be compiled in 1 send.

6. (E) sets are not allowed to be changed if the first game of the day starts within exactly 1 hour/60 minutes and less.

D. LINEUP CHANGES ANSWERS

1. Canceling players more than the required cancel BUT there are still non-playing cancelled players, it can still be replaced as long as the adjusted cancelled players will meet the REQUIRED max cancel.

2. If a player on the Lineup becomes QUESTIONABLE after initially selecting it healthy, the user CAN change players, or even change of all of them, as long as each of the player that will be replaced is NOT YET PLAYING. if the questionable player in the ORIGINAL/first lineup becomes healthy, it will still remain with the 2nd or newly changed Lineup.

3. If a player on the Lineup gets INJURED, you can replace it with players that is not playing yet, you are also freely to change other positions as long as the ones you are going to replace are NOT PLAYING YET.

E. Violations/Issues, with Another Chance

1. If a Quadro team still has not set/not submitted a set and the games has started, you can still set and select players that are NOT ONGOING, refer to the "Updated Salary for late Sets" section.

2. If the set was not submitted or forgot to submit, or if it is only submitted in the group messenger, and vice versa.. send it again, if the games has already started, then replace the ongoing players and replace it with non ongoing.

3. Cancelled more players than the required, just changed the cancelled players until the required amount is reached. This is if the cancelled players are not yet playing, and you can only change the non-playing cancelled players.

F. Violations/Issues, with Automatic Default

1. Canceling players more than the required cancel, if all the players cancelled have already began playing.

2. Overpriced Lineup, this should no longer be an issue, because of the new fantasy website that checks the lineup if it is exceeds the required salary upon clicking "GetLineup", if you are a late set, then compute the required salary for yourself.

G. Updated Salary for late Sets

1-15 Minutes late: 295k

16-30 Minutes late: 290k

30+ Minutes late: 285k

H. Game Results

1. Game Results are only reviseable until 5:00PM, if time passes the deadline, the results are final regardless.

2. If the total team scores are tied, here are the tiebreakers

1st Tiebreaker: Salary used (Adds all the unused salaries of the team, lesser used wins).

2nd Tiebreaker: Offensive rebound stat for CENTERS ONLY (Review and total the offreb of the team's centers).
